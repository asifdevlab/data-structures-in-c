# Data Structures Using C – Lab Programs

This repository contains implementations of fundamental **Data Structures and Algorithms using C** developed as part of the **Semester 2 Data Structures Lab**.

## List of Experiments

| No. | Experiment |
|----|-------------|
| 1 | Find minimum and maximum elements in an array |
| 2 | Menu-based array operations (insert, delete, display) |
| 3 | Search an element in a 2D array |
| 4 | Matrix operations (addition, subtraction, multiplication, transpose) |
| 5 | Singly linked list insertion operations |
| 6 | Singly linked list deletion operations |
| 7 | Stack implementation using array (PUSH, POP) |
| 8 | Factorial and Fibonacci using stack |
| 9 | Linear queue implementation using array |
| 10 | Sorting numbers using Insertion Sort |
| 11 | Sorting numbers using Bubble Sort and analyzing complexity |
| 12 | Comparison of Quick Sort and Selection Sort |
| 13 | Sorting numbers using Merge Sort |
| 14 | Graph traversal using BFS and DFS |
| 15 | Linear search and binary tree traversal |
| 16 | Binary Search Tree operations |
